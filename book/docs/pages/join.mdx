# Join

Live game lobbies are displayed on the game lobby page.

::::steps

### Step one

Click `Join Lobby` button to enter into the waiting room

### Step two

Change your order if desired then valid

### Step three

Click the `Ready` button to indicate you are ready to start

:::warning
Note: The game can be started at any time by the host, players not ready won't be included in the game.
:::
::::

:::tip[Handy Tip]
Click on other players names to show their user info.
:::
