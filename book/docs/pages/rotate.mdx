# Rotate

Rotate the tile appropriately so that it coordinates with the surrounding structures.

If it is not rotated correctly, the player will be unable to confirm (pave) it.

Rotate the tile in clockwise steps by clicking `↩️`.

    <img width="550" src="p-rotate.gif" alt="p-rotate" />
</div>

:::tip[Handy Tips]
Shortcuts - use the key `Q` & `W` keys to rotate the tile in either direction.
:::
