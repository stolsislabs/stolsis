# Create

::::steps

### Step one

Click `Create` button

### Step two

Pop up window appears

### Step three

Type a game name which will be displayed on the lobby list

### Step four

Select a time frame for the game

:::note
Note: if time is set to `0`, the game will be endless.
:::

### Step five

Click `Create` to initiate lobby

### Step six

Returned to game lobby page

### Step seven

Click `Join Lobby` button, change your order if desired then valid
