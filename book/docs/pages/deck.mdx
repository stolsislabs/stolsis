# Deck

## Puzzle mode

The puzzle deck contains **19** unique tiles. 

<div align="center">

:::note
Note: the deck holds **37** tiles in total. Each tile appears an unpredictable number of times (max 6) based on the day's randomised puzzle. The order in which the tiles appear adapts according to the placement of the previous tile. Once the deck has been exhausted, the game is over.
:::
