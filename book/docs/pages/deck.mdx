# Deck

## The Perpetual Puzzle

The Perpetual Puzzle decks contain **38** tiles. These are randomly drawn each day from a set of **72** tiles, **19** of which are unique.

<div align="center">
    <img width="350" src="deck3.png" alt="deck3" />
</div>

:::note
Note: The order in which the tiles appear one after another adapts according to the placement of the previous tile. Once the deck has been exhausted, the game is over.
:::
