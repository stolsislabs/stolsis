## The Perpetual Puzzle

The Perpetual Puzzle decks contains **19** unique tiles.

<div align="center">
    <img width="350" src="deck3.png" alt="deck3" />
</div>

:::note
Note: Each puzzle contains **38** tiles in total. The number of times each tile appears in the deck is randomised daily. The order in which the tiles appear one after another adapts according to the placement of the previous tile. Once the deck has been exhausted, the game is over.
:::
